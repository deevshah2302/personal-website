OC Voltage Automated Sorting Script

UBC Solar — May 2025 to June 2025

Designed and implemented an automated OCV sorting script for 700 18650 Li-ion cells, enhancing pack consistency, manufacturability, and data accuracy through automated voltage-based binning and real-time data logging.

Key Highlights

  • Automated sorting of 700 18650 Li-ion cells using a custom Python script connected to a Keithley Benchtop DMM.
  • Achieved 99.7% voltage classification accuracy, eliminating manual binning errors.
  • Implemented real-time data logging to CSV and automated pass/fail thresholds based on OCV tolerance.
  • Improved pack consistency and balance during module assembly by reducing cell mismatch.
  • Increased testing throughput by 3×, cutting sorting time from hours to minutes per batch.
  • Enabled traceable cell history for long-term degradation tracking and quality control.

Tools Used

Python | Electronic Circuit Design | Lithium-ion Cell Knowledge | NI-Visa
